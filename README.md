OmniDraw — Usage Instructions
Prerequisites

Use a Chromium-based browser (Chrome or Edge recommended).

Ensure JavaScript is enabled.

1. Open Drawaria and Join a Room

Open https://drawaria.online
 in your browser.

Join a drawing room/session where the main drawing canvas is visible.

Important:
If you run the script before joining a room, the script will stop and display an alert instructing you to join a drawing room first.

2. How to Run the Script
Option A (Most Common): Chrome / Edge DevTools Console

Press F12 (or Ctrl + Shift + I) to open Developer Tools.

Navigate to the Console tab.

Paste the entire OmniDraw script into the console.

Press Enter to run it.

Successful launch indicators:

An OmniDraw control panel appears on screen.

A console message confirms that OmniDraw is ready.
3.OmniDraw UI Layout (Tabs)

The OmniDraw panel is organized into the following tabs:
Image
Path
Bot
Placement
Preview
Guide
Settings
Standard Workflow (Recommended Order)
Step 1 — Image Tab (Load Your Image)

Open the Image tab.

Load or select the image you want OmniDraw to reproduce.

Once loaded:

OmniDraw can generate drawing commands.

Progress messaging becomes active.

(Before loading, the UI will prompt you to load an image.)

Step 2 — Path Tab (Generate Strokes)

Open the Path tab.

Generate/build the drawing path (the ordered strokes OmniDraw will send to the canvas).

If the result is:

Too slow

Too detailed

Too heavy
You can later adjust resolution, stroke density, or color limits in the Settings tab (see Section 7).

Step 3 — Placement Tab (Fit to Canvas)

Open the Placement tab.

Adjust positioning controls:

Scale

Offset X

Offset Y

Alignment

These settings determine how the drawing is positioned and sized on the Drawaria canvas.

Step 4 — Preview Tab (Verify Before Drawing)

Open the Preview tab.

Play the preview animation to confirm:

The position is correct.

The path sequence looks reasonable.

The level of detail matches your expectations.

Available controls include:

Play / Pause / Reset

Preview duration setting

Once the preview looks correct, you are ready to proceed with actual drawing execution.
